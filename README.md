# AQMA Pak Airways ✈️

AQMA Pak Airways is a simple airline website project developed using **HTML, CSS, and JavaScript** for the frontend, and **PHP with MySQL** for the backend.  
The project is designed to provide users with an attractive and responsive interface for exploring airline services, flight information, destinations, and booking-related details, while administrators can manage flights, users, and bookings securely.

---

## 🚀 Features
* Clean and user-friendly design
* Responsive webpage layout
* Navigation menu for easy browsing
* Flight and destination information sections
* Modern styling using CSS
* Structured and semantic HTML
* Interactive elements with JavaScript
* User Registration & Login (PHP + MySQL)
* Admin Dashboard for managing flights and users
* Secure Session Management (Login/Logout)
* Database Integration with MySQL

---

## 🛠️ Technologies Used
- **Frontend:** HTML5, CSS3, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server:** XAMPP / Laragon  

---

## 🎯 Purpose
This project was created to practice and demonstrate **full‑stack web development skills**, including:
- Webpage structure and styling  
- Layout design and responsive user interface development  
- Dynamic interactivity with JavaScript  
- Backend development with PHP  
- Database management using MySQL
